# Copilot Instructions

## Design Guide

When designing UI or visual experiences in this repository, follow these principles:

- Prefer a strong, coherent visual identity rather than generic interface patterns.
- Use CSS utility classes and minimal markup to keep pages easy to maintain.
- Favor distinctive typography and color palettes that support the project tone.
- Add motion selectively: focus on meaningful transitions, hover effects, and entrance animations.
- Keep layout responsive and accessible across modern browsers and screen sizes.
- Use the existing stylesheet conventions in `socops/src/main/resources/static/css/app.css`.
- When implementing retro or thematic styles, reinforce them with both background texture and component polish.

## Notes

- This repository uses Thymeleaf templates under `socops/src/main/resources/templates/`.
- Keep new styles self-contained where possible, and avoid adding large external dependencies.
